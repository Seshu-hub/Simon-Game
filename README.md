
Simon Game is a web-based memory game where the computer flashes a sequence of colors, and the player must repeat the sequence in the correct order. Each level adds a new color to the sequence.

## How to Play
1. Open `index.html` in a web browser.
2. Press any key to start.
3. Watch the color flash and click the same button.
4. Repeat the sequence as it grows with each level.
5. Wrong click? Game over, press any key to restart.

 ## Technologies
